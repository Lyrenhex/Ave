About Ave

Ave is a new IRC (internet relay chat) client designed according to the Google Material Design style guidelines. Ave aims to modernise the increasingly dated IRC chat system, by using a sleek, modern design (as opposed to the many IRC clients that have a design much like Windows XP applications), along with introducing features which are more commonly found on later chat systems (such as a clearer reply system).

Ave also serves to add to the free software community, as Ave is free - both as in cost, and in freedoms.
