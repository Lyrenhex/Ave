Ave Help and Information

It should be noted that this information is intended for the end user. In the event that you are a maintainer of the project, or you're just super interested in more detailed information regarding Ave, you should check the technical help document.

If you are stuck with using Ave, please see the contents of this document for assistance.

Connecting to a server

Ave intends to simplify, as much as possible, the task of connecting to servers. That said, there is information that Ave requires to connect you to IRC. When you launch Ave, you should see a connection screen. There will be three fields: Server Address; Server Port; and Nickname. There are more fields (marked as advanced), but these should not be touched unless you're sure you know what you're doing.

The Server Address should be the IP Address of the intended server; by default this field points to the Freenode IRC network (irc.freenode.net).

The Server Port field should be the port that the intended server operates on. As most IRC servers operate on port 6667, this defaults to 6667.

The Nickname field must be manually set by yourself. Your nickname is your username on the IRC server, and is used to identify you (it is shown publicly). No two users on IRC may ever share a nickname; if your desired nickname is already in use, Ave will add a number to the end of it for you. Nicknames may be changed once you are connected. Nicknames should not exceed 9 characters. However, due to many servers supporting nicknames up to 30 characters, Ave will permit nicknames of lengths up to 30 characters.

Once connected, you will enter an isolated "Sys" channel. If you never receive the message that you have connected to the server, and the channel hangs on "Connecting...", then Ave has failed to establish a connection. You may wish to restart Ave and check your Server details.

Please note that connection details will save once you attempt to connect to a server.
